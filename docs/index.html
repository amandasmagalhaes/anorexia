1 Objetivo

Estimar a prevalência e características sociodemográficas associadas ao risco de anorexia.

2 Métodos

As análises utilizaram dados da terceira onda do Estudo Longitudinal da Saúde dos Idosos Brasileiros (ELSI-Brasil). Mais informações sobre o estudo estão disponíveis no site do ELSI-Brasil.

4 Organização das variáveis

A amostra inicial incluiu 10.773 indivíduos. Foram excluídos 153 participantes com resposta “não se aplica” ou “não sabe/não respondeu” para anorexia e 2.176 indivíduos com menos de 60 anos de idade, resultando em uma amostra analítica final de 8.444 participantes.

calc_prev <- function(var, label, desenho) {
  prev <- svyby(
    ~ I(snaq_cat == "≤14"),
    as.formula(paste0("~", var)),
    desenho,
    svyciprop,
    vartype = "ci",
    method = "logit",
    na.rm = TRUE
  )
  p_val <- svychisq(
    as.formula(paste0("~n73_6 + ", var)),
    desenho
  )$p.value
  categoria <- as.character(prev[[var]])
  tibble(

      paste0("snaq_cat ~ ", v)
    ),
    design = desenho,
    family = quasibinomial()
  )
  p_global <- regTermTest(
    modelo,
    as.formula(
      paste0("~", v)
    )
  )$p
  res <- tidy(
    modelo,
    conf.int = TRUE,
    exponentiate = TRUE
  ) |>
    filter(term != "(Intercept)")
  ref <- levels(df[[v]])[1]
  bind_rows(
    tibble(
      Caracteristica = labels[[v]],
      OR_IC95 = "",
      p_valor = ifelse(
        p_global < 0.001,
        "<0.001",
        sprintf("%.3f", p_global)
      ),
      destaque = p_global < 0.20
    ),
    tibble(
      Caracteristica = paste0("  ", ref, " (ref.)"),
      OR_IC95 = "1.00",
      p_valor = "",
      destaque = FALSE
    ),
    tibble(
      Caracteristica = paste0(
        "  ",
        gsub(
          paste0("^", v),
          "",
          res$term
        )
      ),
      OR_IC95 = sprintf(
        "%.2f (%.2f–%.2f)",
        res$estimate,
        res$conf.low,
        res$conf.high
      ),
      p_valor = "",
      destaque = FALSE
    )
  )
